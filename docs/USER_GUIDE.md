# Constella 用户指南

## 1. 适用范围

- 客户端形态：Web 与 Electron 桌面端（安装包、Portable）。
- 典型用户：协作成员、房主、管理员。
- 目标：帮助你从首次连接到日常协作与导出。

## 2. 快速开始

### 2.1 准备工作

- 确认服务端已启动并可访问。
- 准备服务端地址，例如 `http://127.0.0.1:3000` 或 `http://192.168.x.x:3000`。
- 局域网协作场景下，确认防火墙放行相关端口。

### 2.2 启动客户端

- Web：打开部署地址或本地开发地址。
- Desktop 安装版：通过安装程序安装后启动。
- Desktop Portable：解压后直接运行可执行文件。

### 2.3 首次连接服务端

1. 在首页输入服务端地址。
2. 点击连接。
3. 连接成功后进入登录页。

桌面端支持局域网自动发现：

- 单击发现项可预览地址。
- 双击发现项可直接连接。

## 3. 登录与注册

### 3.1 登录

1. 输入密码。
2. 登录成功后进入房间页。

### 3.2 注册

当账号不存在时，系统会进入注册流程：

1. 输入密码。
2. 输入确认密码。
3. 仅当两次密码一致时，允许注册。

说明：当前版本不再在“账号不存在”时直接自动注册。

## 4. 房间协作流程

### 4.1 房间列表

- 支持查看全部房间、我的房间、已加入房间。
- 可对房间进行收藏和快捷进入。

### 4.2 创建房间

1. 点击创建房间。
2. 填写房间名与可选描述。
3. 选择公开或私密。
4. 私密房间可设置访问密码。

### 4.3 加入房间

- 公开房间可直接加入。
- 私密房间需输入房间密码。

### 4.4 权限与角色

- 房主：完整管理能力。
- 管理员：可协助管理成员与资源。
- 成员：日常协作编辑。

## 5. 画布基础操作

### 5.1 导航与视图

- 平移：拖动画布或使用平移工具。
- 缩放：鼠标滚轮缩放。
- 选中：单击节点或框选。

### 5.2 节点与连线

- 新建节点：通过工具栏创建不同类型节点。
- 编辑节点：双击节点打开编辑器。
- 连线：从起点节点连接到目标节点。

### 5.3 常用面板

- 右侧面板：节点属性、结构、资源等。
- 成员面板：查看在线成员状态。
- 聊天面板：房间内即时沟通。

## 6. 节点编辑与内容能力

### 6.1 文本与 Markdown

- 支持普通文本与 Markdown 渲染。
- Markdown 编辑器支持常见格式与命令。

### 6.2 数学公式与代码块

- 支持 KaTeX 数学公式渲染。
- 支持多语言代码块高亮。

### 6.3 Mermaid 图表

- 支持常见 Mermaid 图类型。
- 适合流程图、时序图、思维导图等表达。

## 7. 资源与快照

### 7.1 资源上传

- 支持上传图片资源并插入画布。
- 资源权限受房间角色控制。

### 7.2 快照管理

- 可创建当前画布快照。
- 可在需要时恢复历史快照。

## 8. 导出能力

### 8.1 画布导出

- 支持导出 JSON、PNG、SVG。

### 8.2 文档导出

- 支持导出 Markdown、Text、PDF（视运行环境而定）。

## 9. 设置与个性化

- 主题切换：浅色/深色。
- 语言切换：中文/英文。
- 账号信息：昵称、头像、用户标识。
- 编辑器与协作偏好：可在设置面板调整。

## 10. 常见问题排查

### 10.1 无法连接服务端

- 检查服务端是否运行。
- 检查地址与端口是否正确。
- 检查本机网络与防火墙策略。

### 10.2 局域网发现不稳定

- 优先使用 IPv4 地址连接。
- 刷新发现列表或手动输入地址。
- 必要时重启客户端与服务端刷新缓存。

### 10.3 桌面端启动异常

- 确认目标机器安装了 Node.js。
- 检查日志目录与系统权限。

## 11. 反馈与支持

- 使用中遇到问题，可在仓库提交 Issue。
- 提交问题时建议附带：运行环境、复现步骤、日志截图。
