# Constella v1.0.0 使用教程

## 1. 运行前准备
- 操作系统：Windows（当前桌面产物以 Windows 为主）。
- Node.js：建议 `>= 20`。
- 网络：局域网协作时，确保主机与客户端可互相访问。

## 2. 启动软件
1. 解压或复制 `win-unpacked` 目录到目标机器。
2. 双击运行 `Constella.exe`。
3. 首次启动若被系统拦截，请允许应用通过防火墙。

## 3. 首次连接
1. 在首页填写后端地址（例如 `http://127.0.0.1:3000` 或局域网地址）。
2. 登录或注册账号。
3. 创建房间或加入已有房间。
4. 进入画布后可进行节点创建、编辑和连线操作。

## 4. 画布基础操作
- 平移：按住空格 + 鼠标拖动，或切换到平移工具。
- 缩放：鼠标滚轮缩放。
- 节点：单击选中，双击打开编辑。
- 连线：切换连线工具后，按起点节点拖至目标节点。
- 多选：框选或使用组合键进行追加选择。

## 5. 局域网协作要点
1. 后端监听 `0.0.0.0`（默认端口 `3000`）。
2. 前端通过局域网地址访问（如 `http://192.168.x.x:5173`）。
3. 前端 API / WS 目标不要写死 `localhost`，应使用主机局域网地址。
4. 放行防火墙端口（常见为 `3000`、`5173`）。

## 6. 常见问题
### 启动报错 `spawn node ENOENT`
- 原因：目标机器缺少 Node.js。
- 解决：安装 Node.js（建议 `>= 20`）后重试。

### 无法连接后端
- 检查后端是否已启动。
- 检查后端地址和端口是否正确。
- 检查防火墙和代理策略。
