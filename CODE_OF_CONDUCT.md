# Code of Conduct

## Our Pledge

We want the Constella web repository to be welcoming, collaborative, and safe for everyone. Contributors, maintainers, and community members are expected to interact with respect, empathy, and professionalism.

## Expected Behavior

- Be respectful in discussions, reviews, and issue threads.
- Assume good intent and give constructive feedback.
- Share context clearly when reporting bugs or proposing changes.
- Help keep discussions focused on improving the project.

## Unacceptable Behavior

- Harassment, hate speech, discrimination, or personal attacks.
- Trolling, intimidation, or deliberately disruptive behavior.
- Publishing private information without consent.
- Repeatedly ignoring maintainer guidance in issues, discussions, or pull requests.

## Scope

This policy applies in:

- GitHub issues, pull requests, and discussions
- Project documentation and review comments
- Community spaces directly related to this repository

## Enforcement

Maintainers may remove comments, close discussions, or restrict participation when behavior violates this Code of Conduct.

For serious or repeated issues, contact the repository maintainer privately through GitHub. Please include links, screenshots, or other relevant context when possible.
